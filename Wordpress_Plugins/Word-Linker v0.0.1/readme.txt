== Description ==

Automatically link user defined words or sentances to a URL. Only the first match is replaced on each page.

== Installation ==

1. Upload the whole `word-linker` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In WordPress admin go to 'Appearance' -> 'Settings' -> 'Word Linker' to specify the replacements.
